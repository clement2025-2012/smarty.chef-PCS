Smarty-Chef.PCS collects minimal data necessary for functionality:

We use your data to:

Smarty-Chef.PCS is crafted with passion by Clement, combining modern web technologies with smart recipe generation.

  • 🔍 Smart ingredient selection with Indian spices
  • 🤖 AI-powered recipe generation via API Ninjas
  • 🥗 Dietary preference filtering
